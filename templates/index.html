What is this website?

This is a web application that I've created for MS587, Databases and Web Development at the University of Advancing Technology. The purpose of this application is to create a Web Application that will store books that a user can read and incorporate text-to-speech to make a primitive audiobook. The user will be able to choose which books they want to read and go from there.

A benefit of web applications is that since they're hosted online, they do not need to be downloaded like a traditional app. Users will be able to interface with your program fully from within their web browser, freeing up space on hard drives and is generally not reliant on a user's hardware.

How To Use This Site

In order to use this website, click on the Book Catalogue tab and read through the available books in the catalogue. Once you find the perfect book, click the link that says "Go to Book Page". On that page, you'll have the option to download the book, read the book directly on the website or even use AI Text-To-Speech to start the Audiobook!
